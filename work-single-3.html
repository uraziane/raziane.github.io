Greeting Cards & Gift Dressings

UK Greetings

YCN Student Awards 2016/17

Discover Autumn love where each greeting card and gift dressing tells a story. This project explores a collection of greeting cards and gift dressings for UK Greetings. It includes greeting cards, notecards, notepads, gift bags and wrapping paper.

The chosen theme is Geometric + Autumn. The cards were designed for the upcoming autumn for target audience aged 25-35 years. The set consists of different occasions such as birthdays and anniversaries. Assorted gift dressings were designed to create a unified set and create a memorable user experience.
